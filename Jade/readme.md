# jade

[jade](http://jade-lang.com/)

node.js 가 설치되어 있어야 한다.


```
// jade 설치
npm install jade -g

// index 파일 자동 컴파일
jade -w index.jade

// 코드를 보기좋게 나열해서 컴파일
jade -w --pretty index.jade

// 두개의 파일을 컴파일 할때
jade -w --pretty index.jade about.jade
```
