## id
- 탁타표기법(카멜케이스 cameCase)을 사용한다.단어를 연결할때 연결되는 첫글자는 대문자로 표기.
- 하이픈(`-`) 과 언더바(`_`)를 사용하지 않는다.
- 첫글자는 반드시 소문자로 표기한다.
- 의미있는 아이디를 사용한다.(예: )



## class
- 단어를 조합해서 만들경우 하이픈(`-`)를 사용한다.
- 언더바(`_`)를 사용하지 않는다.
